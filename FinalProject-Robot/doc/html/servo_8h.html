Detailed Description

Author
Jake Drahos

Blocking/NonBlocking Servo API

This API uses a number of conversion functions and a core set function, SERVO_SetNonBlocking(). It also contains several wrapper macros that wrap the conversion and call to the set function in a single statement.

uint16_t SERVO_SetNonBlocking ( uint16_t  pwm)

Set the PMW value and return recommended msec delay

This function wil probably not be called directly, since the PWM value required for this function must be calculated with one of the conversion functions. It is wrapped by several macros.

This function is asynchronous and returns immediately, but gives a recommended msec delay.
