Detailed Description

Author
Jake Drahos This contains the USART functions. The USART driver is completely asynchronous. User-space functions only manipulate a collection of ringbuffers. The ringbuffers are only accessed single-ended by either the ISR or userspace, so they are interrupt-safe. Because it is asynchronous, it is FAST.
