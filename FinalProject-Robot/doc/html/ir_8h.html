uint16_t IR_GetData ( )

Get the most recent IR measurement

The IR API is completely asynchronous. The ADC is continuously sampled. IR_GetData() simply returns the most recent sample. IR_GetDataBlocking() blocks until new data is available before returning.

Returns
Raw ADC value of IR sensor
uint16_t IR_GetDataBlocking ( )

Block until new data is available, then return it.

Similar to IR_GetData(), but waits for new data before returning.

Returns
Raw ADC value of IR sensor

Initialize IR This must be called to set up the interrupts before any other IR functions are called. If a RAM lookup table is being used, this function will also populate the lookup table.
